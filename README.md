# QuarkSearch-快搜

![QuarkSearch](/QuarkSearch.jpg)

QuarkSearch，一款基于langchain框架的多agent搜索增强工具

## 快搜能干什么

首先将你脑海中想搜索的模糊的概念交给快搜，快搜对其进行增强（列出所有可能的结果）。接着，快搜将在各个平台进行搜索，最后把最符合你想搜索的结果返回给你，这就是快搜。
